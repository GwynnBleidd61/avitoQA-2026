# Задание 1. Баги на скриншоте

## Список багов

1. **P1 - В поисковой выдаче по Москве отображается объявление с локацией «Новосибирск».**  
   Пользователь видит страницу «Котята в Москве», однако в выдаче присутствует объявление из другого города.

2. **P1 - В выдаче отображается объявление за 500 ₽ при установленном минимальном фильтре цены 1000 ₽.**  
   Результаты поиска не соответствуют установленному фильтру по минимальной цене.

3. **P1 - Переключатель «Сначала из Можайска» противоречит выбранной локации «Москва».**  
   В правом верхнем углу указана локация «Москва, район, метро», но рядом с сортировкой отображается переключатель с другим населённым пунктом, что создаёт противоречие в фильтрации.

4. **P2 - В фильтре возраста для категории котят отображается значение «Щенок».**  
   На странице с котятами в блоке фильтра «Возраст» используется некорректное значение, относящееся к собакам, а не к кошкам.

5. **P2 - При наведении на иконку избранного отображается некорректная подсказка «Не показывайте мне похожие объявления».**  
   Подсказка не соответствует назначению иконки сердечка. Для этого элемента ожидается действие, связанное с добавлением в избранное.

6. **P2 - На странице одновременно отображается общее количество результатов «19» и кнопка «Показать 8 объявлений», что даёт пользователю противоречивую информацию о количестве найденных объявлений.**  
   Пользователь видит два разных количества результатов на одном экране, из-за чего не понимает, сколько объявлений действительно найдено по текущим фильтрам.

7. **P2 - В выдаче по котятам присутствует нерелевантный товар: плюшевый котёнок.**  
   В результатах поиска по живым котятам отображается товар, не соответствующий тематике выдачи.

8. **P3 - В верхнем меню допущена опечатка: «Коръера в Авито» вместо «Карьера в Авито».**  
   На странице присутствует орфографическая ошибка в одном из пунктов верхнего меню.

9. **P3 - В хлебных крошках нарушен порядок категорий.**  
   Сейчас отображается путь: «Главная > Животные > Персидская > Кошки», тогда как корректная логика категорий должна быть: «Главная > Животные > Кошки > Персидские».
